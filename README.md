# Restaurant Sales Performance Dashboard

![Dashboard Preview](Restaurant-Sales-Dashboard.png)

## Overview
This project analyzes restaurant sales data using Power BI to identify revenue trends, top-performing products, and category performance.

## Tools Used
- SQL
- Power BI
- Excel

## Key Insights
- Total Revenue: 5.97K
- Total Orders: 500
- Average Order Value: 11.94
- Top Category: Main
- Top Product: Pizza

## Dashboard Features
- KPI metrics (Revenue, Orders, Average Order Value)
- Product and category performance analysis
- Monthly revenue trend visualization
- Interactive category filter

## Purpose
This project demonstrates data analysis, dashboard development, and business intelligence skills.
